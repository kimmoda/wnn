Installation

  1. Unzip the application.zip file and open the "music" folder
  2. Open the project with Android Studio. Note : (If Gradle Update Prompt appears, Do not Update it)
  3. Now navigate to CordovaLib\src\org\apache\cordova\ConfigXmlPaser
  4. Now replace ' private String launchUrl = "Your Url Here"; ' and Now Scroll down to bottom, replace ' launchUrl = "Your Url Here";' with your own url
  5. Now Navigate to res/xml/config.xml and replace " Your Url Here " with your own url
  6. Now Navigate to res/values/string.xml and replace "your App name","Your Purchase Code","Your Websiteurl"
  7. Note : You can get "Purchase Code" from your Envato Download Page. Go to this url : https://codecanyon.net/downloads Click on Download Button and Download "License Certificate and Purchase code (PDF)" file.
  8. You can also change the APP icon and SPLASH screen in res folder (replace with your app graphics)
  9. After finishing the above steps, you can generate the app by clicking on Build -> Generate APK
  10. Your APP will generated in the ..\build\outputs folder of the project
  11. If you have any problem, Contact me by leaving a message on : https://codecanyon.net/user/scriptwriter2017


    Push Notification :


  • Go to OneSignal website : https://www.onesignal.com
  • Sign in and Create a app with oneSignal
  • Next follow the intructions in this website : https://documentation.onesignal.com/docs/generate-a-google-server-api-key
  • After that go to OneSignal dashboard --> APP Settings, and get your oneSignal App Id under Keys & ids tab
  • Now open the project in Android Studio, Navigate to "build.gradle(Module:WebToNative)"
  • Now find for keyword "Your OneSignal App ID" and replace it with your Original App ID from OneSignal Dashboard
  • Now Build The Apk and Install it on your Android Phone
  • Go to OneSignal Dashboard and Click on "New Message" Tab & Follow the instructions in the Dashboard to send Push Notifications


Note That : Dont change the package name (if you change the package name, it will show error). if you want to change the package name --> contact me : https://codecanyon.net/user/scriptwriter2017
